# Shopping Dataset Analysis

## Objective

The objective of this assignment is to perform exploratory data analysis (EDA), data cleaning, and generate insights from the shopping dataset using Python and Pandas.

## Dataset Used

* Combined_dataset.csv

## Steps Performed

### Step 1: Load Data

* Imported required libraries
* Loaded the dataset
* Checked dataset shape and column names

### Step 2: Understand Data

* Checked data types
* Identified missing values
* Used info() and describe() for understanding the dataset

### Step 3: Data Cleaning

* Converted price columns to numeric format
* Handled missing values
* Removed duplicate records

### Step 4: Feature Engineering

* Created price_difference column
* Created popularity_metric column

### Step 5: Analysis

* Performed univariate analysis
* Performed bivariate analysis
* Conducted category-level analysis

### Step 6: Visualization

* Created histogram
* Created bar chart
* Created boxplot

### Step 7: Insights

* Summarized key findings
* Discussed business implications

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
