# E-Commerce Sales Database Analysis

## Overview

This project focuses on analyzing the Sample Superstore dataset using SQL. The dataset was imported into MySQL, explored, and analyzed using different SQL queries to understand sales performance and business trends.

---

## Objective

- Load the dataset into MySQL
- Explore the table structure
- Apply filtering using WHERE
- Perform aggregations using GROUP BY
- Find top-performing products and customers
- Analyze monthly sales trends
- Validate data quality

---
